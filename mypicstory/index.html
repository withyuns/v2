Introduction

Intro

All motivation comes from my experience (photography as hobby, blogging experience), and I have many professional photographers and bloggers as human resources! - Good news!

Taking a picture is my favorite hobby, and I always uploads all pictures in existed blog platform after taking a shot. However, it is very inconvenient not to manage pictures using the platform. Sometimes I just want to show pictures only (Like, Instagram), but also I want to order pictures based on filters. (or show favorite pics only - owners selected). Instagram doesn’t have it!

As a blogger, I need a storyline with multiple pictures. For example, let say I travel to New York, I want to post a travel story such as… “I took a subway and I lost my way, and I found best restaurant ever by chance!”.. This is how I share my all experience with others using my blog!

Ok, I need combined platform! How..?

    • Prototypes & Design

    First rending page

    This is first page of the web app. The app has two modes.

  • Story mode - For Blogger
  • Gallery mode - For Photographer
    • (I am planning to expand all other features in the future.)
    Depends on mode users chose, our app shows two different versions.


    1. Main page

    - Category

    When users upload a file/files, user can choose category (If there is no, users can create new category). By default, users can see all pictures depends on category.


    - Filtering from main page

    For now, Pictures can be sorted based on date(posted/shoot), a number of views and favorited by owner. More filters can be added in the future.


    2. Story Mode

    This page only show all list of stories. Users can choose any picture as thumnail when they make the story OR when they modify the story. Following by thumnails, short description will be shown.


    3. Gallery Mode

    Just like other pictures app, this mode displayed all images. All image is clickable, so users can enlarge the image by clicking it.


    Uploading page

    When users upload image file, users can choose category OR users can create a new category. Users can choose a date shoot (otherwise, uploading date will be set). Users can put a title and description.
    If users want to make a story, then just click a "Make a Story" button. Then users can see other window that can include multiple images, and also can give title and description for the story.
